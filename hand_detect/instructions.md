
## Deployment

To deploy this part of the project, first

Change directory into the folder. 

a. Create a virtual environment

  - on `Python` > 3.x
```
  python3 -m venv myenv
```
  - if `Python` < 3.x or alias set
```
  python -m venv myenv
```
b. Activate the virtual environment(venv)
  - on MacOS/Linux
```
  source myenv/bin/activate
```
  - on Windows (cmd)
```
  myenv\Scripts\activate
```

c. Run `hand.py`,

```
  python3 hand.py
```
or 
```
  python hand.py
```
